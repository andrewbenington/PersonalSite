    The Project

    This program built for my CS 126 in the fall of my freshman year. It can import any MIDI music file for the player to play along with GarageBand-style. While all of a file's tracks would be audibly played, the user could pick which one would be the one on screen. I wrote an algorithm to decide which key each note would play on that took into account the notes played before and after it, with the goal of keeping the keys varied as much as possible. Using the MIDI file's metadata, I predicted which tracks were percussion and disabled them, otherwise they would have added undesirable noise.
